# kleidi-task Documentation

Welcome to the kleidi-task documentation. Here you'll find everything you need to install, configure, and use kleidi-task for your development workflow.

## Getting Started

1. [Installation](installation) — Install the `klt` binary
2. [CLI Reference](cli) — Learn the commands
3. [MCP Integration](mcp-usage) — Connect your AI assistant

## Interfaces

kleidi-task provides 4 ways to interact with your tasks:

| Interface | Best for |
|-----------|----------|
| [CLI](cli) | Quick terminal usage |
| [REST API](rest-api) | Automation and scripts |
| [MCP Server](mcp-usage) | AI assistants (Claude, Cursor) |
| Web UI (`klt serve`) | Visual kanban board |
